## Getting Started

### Clone the Repository and Install

- Use Git to clone the repository to your local machine and download dependencies:

  ```bash
  git clone https://github.com/therealraffi/MLatUVA_Website.git
  cd MLatUVA_Website/sigai_site
  npm install
  npm audit fix # incase you get any issues
  ```

  Download `.env.production` and `.env.development` from [here](https://drive.google.com/drive/folders/1yF0BBwAbfX3I213xj03jEOamxarLbaNX) and place them in `SIGAIatUVA_Website/sigai_site` (see directory tree above)

### Start Development Server

- To start the development server, run:

  ```bash
  npm run dev
  ```

  and follow subsequent instructions

### Build for Production

- To start the development server, run:

  ```bash
  npm run build
  ```
